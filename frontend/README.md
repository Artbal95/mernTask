# React + TypeScript + Vite + Eslint

## Frontend

### Version

```
$ node >= 16
$ typeScript >= v4
```

### Environment
```bash
# Create .env.local file with this parameters
VITE_APP_BASE_URL
VITE_APP_BASE_PREFIX
```

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# Run
$ yarn run dev

# Run to build
$ yarn run build
```
